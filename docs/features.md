# Feature Brainstorm List (Not all will be implemented)

## Core Gameplay
- Player movement (run/jump/dash) + basic physics
- Basic combat (melee/range)
- Enemy AI (patrol, chase, attack)
- Boss fight at the end of each stage
- Stage progression (finish stage -> next stage)

## Levels/Stages
- Simple stage layout with platforms/obstacles
- Few enemies types

## UI/Feedback
- Health bar + damage feedback (flash, knockback)
- Basic main menu + restart
- Win/Lose screens
- Simple sound effects (hit, jump, boss)

## Player Upgrades / Rewards (Stretch)
- Pick up coins/items
- Simple upgrades (more HP/damage)
  
## Polish (Stretch)
- Better boss patterns / phases
