# MVP (Minimum Viable Product)

Goal: A playable 2D game where the player clears 1 stage and defeats 1 boss to win.

## MVP Features (Must Have)
1. Player controller
   - Move left/right
   - Jump/dash
2. Player combat
   - Basic attack that can damage enemies
3. Basic enemy
   - Simple AI: patrol or chase and deal damage
4. Boss enemy
   - Bigger HP
   - At least 2 attack patterns
5. One playable stage
   - Start area -> path -> boss room
   - Win condition: boss defeated
6. UI basics
   - Player health display
   - Win screen + restart button/continue (if more stages)
   - Lose screen + restart button

## Out of Scope for MVP (If there is enough time)
- Multiple stages
- Upgrades
- Fancy animations/polish
